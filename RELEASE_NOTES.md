# v1.6.0 更新说明

这版调整通知栏和桌面小组件的职责：通知只显示状态，操作集中到小组件。

## 变化

- 移除通知栏展开态，不再设置 `bigContentView`。
- 移除通知栏里的 `解锁 / 停充 / 停止` 操作按钮。
- 通知栏固定为一栏两排状态展示：第一排 App 名称，第二排当前状态与操作摘要。
- 桌面小组件改为主操作面板，提供 `解锁 / 停充 / 停止` 三个按钮。
- 小组件标题区域可点击进入 App 设置。
- 小组件 UI 调整为紧凑 Bento/Material 风格控制面板。

## 关于覆盖安装

从 `v1.5.1` 开始，GitHub Release 里的 APK 使用同一把固定签名 key，后续版本可以直接覆盖安装升级。

如果手机上已经安装的是 `v1.5.0` 或更早版本，由于旧版 GitHub Actions 每次构建会生成随机 debug 签名，第一次升级到 `v1.5.1` 时 Android 仍可能提示签名不一致。这种情况需要先卸载旧版，再安装 `v1.5.1`。之后再升级新版本就不需要卸载了。
