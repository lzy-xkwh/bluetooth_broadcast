# v1.6.1 更新说明

这版继续压缩通知栏样式，让它更接近普通系统通知：一行显示、弱化存在感，但保留关键状态。

## 变化

- 通知栏去掉 App 标题行，只保留一行“状态 + 操作摘要”。
- 通知栏右侧状态 TextView 保留，用来快速看 `待命`、`广播中`、`不可用`。
- 通知栏图标从 48dp 调整为 40dp，整体高度从 72dp 调整为 64dp。
- 通知栏字体改为普通 14sp，降低与其他通知相比的突兀感。
- 通知栏仍然不放 `解锁 / 停充 / 停止` 按钮，操作继续集中在桌面小组件。

## 关于覆盖安装

从 `v1.5.1` 开始，GitHub Release 里的 APK 使用同一把固定签名 key，后续版本可以直接覆盖安装升级。

如果手机上已经安装的是 `v1.5.0` 或更早版本，由于旧版 GitHub Actions 每次构建会生成随机 debug 签名，第一次升级到 `v1.5.1` 时 Android 仍可能提示签名不一致。这种情况需要先卸载旧版，再安装 `v1.5.1`。之后再升级新版本就不需要卸载了。
